# Direct Patch Mode - AI 增强版

## 🎯 功能概述

**Direct Patch** 是一个智能图像合成模式，它将元素直接"打补丁"到目标图像中，然后使用 AI 进行后处理融合，确保：

- ✅ **自然融合** - 新元素与周围环境无缝整合
- ✅ **光影一致** - 自动匹配场景的光照方向和强度
- ✅ **风格统一** - 保持整体风格和色彩和谐
- ✅ **细节保留** - 保持高质量细节和纹理
- ✅ **角色一致性** - 如果有角色，保持面部、发型、姿势不变

## 🔄 工作流程

### 1. 初始合成阶段
```
用户上传图片 → 裁剪元素 → 移除背景 → 直接合成
```

- 用户选择要复制的元素区域
- 系统自动移除背景（使用 RMBG 模型）
- 将元素直接粘贴到目标位置

### 2. AI 融合阶段
```
合成图片 → AI 分析 → 光影调整 → 风格匹配 → 输出融合结果
```

- **光照匹配** - 分析场景光源，调整新元素的亮度和阴影
- **阴影生成** - 根据光源方向添加真实阴影
- **色温调整** - 匹配环境的色彩温度
- **边缘融合** - 自然混合边缘，同时保持细节清晰
- **风格统一** - 确保视觉风格（写实/卡通等）一致

### 3. 水印去除阶段 ✨ 新增
```
融合图片 → 检测水印 → AI 修复 → 输出无水印结果
```

- **智能检测** - 自动识别图片中的水印位置
- **AI 修复** - 使用 AI 模型智能填充水印区域
- **纹理匹配** - 匹配周围的纹理和颜色
- **无缝去除** - 使水印去除后不留痕迹
- **内容保留** - 完全保留其他所有内容（角色、背景等）

## 🚀 使用方法

### 步骤 1: 选择 Direct Patch 模式
在编辑参数面板中，点击 **"Direct Patch"** 按钮。

### 步骤 2: 上传图片
- **Element Image** - 包含要复制元素的图片
- **Base Image** - 目标背景图片

### 步骤 3: 裁剪元素
使用 Element Crop 工具精确选择要复制的元素区域。

### 步骤 4: 绘制掩码
在 Canvas Editor 中标记要粘贴的位置（白色区域）。

### 步骤 5: 点击 Patch
点击 **"Patch"** 按钮开始处理。

### 步骤 6: 等待 AI 处理
系统会依次执行：
1. 背景移除
2. 元素合成
3. **AI 融合和谐化** ✨
4. **智能水印去除** ✨ 新增

- 风格相似的图片融合效果更自然
- 复杂光影场景可能需要更长处理时间

## 🎯 智能裁剪改进 (v2.2)

### 问题背景
在之前的版本中，用户反馈使用画笔工具选择元素时，裁剪区域不够准确。例如：
- 用户没有选中手部，但最终裁剪包含了手
- 意外画到的边缘区域被包含在裁剪中
- 裁剪框过大，包含了不需要的背景

### 解决方案

#### 1. 离群像素检测
系统现在会分析所有画笔涂抹的像素，自动识别"离群区域"：
- 将画布分成 20x20 像素的区域网格
- 计算每个区域的像素密度
- 识别密度低于平均值 15% 的稀疏区域为"离群点"
- 自动排除这些区域，只保留主要选区

**效果**: 如果用户意外画到了远离主选区的小区域（如手部），系统会自动忽略这些区域。

#### 2. 自适应内边距
在确定最终裁剪框后，系统会：
- 计算选区大小的 2% 作为内边距（1-3 像素）
- 向内收缩裁剪框，避免包含边缘伪影
- 确保裁剪更紧凑、更精确

**效果**: 排除画笔边缘的模糊像素和意外涂抹。

#### 3. 详细日志输出
控制台会显示：
```
[ElementCropper] Outlier removal reduced area by 12.3%
[ElementCropper] Crop region (display, with padding): 145 203 156 198
[ElementCropper] Crop region (natural): 580 812 624 792
```

这有助于调试和理解裁剪逻辑。

### 使用建议
- ✅ 画笔工具：专注于主要选区，不用担心小范围的意外涂抹
- ✅ 如果看到"Outlier removal"日志，说明系统成功过滤了离群区域
- ⚠️ 如果裁剪仍不准确，尝试使用矩形选择工具获得精确控制
- ⚠️ 对于复杂形状，多次小范围涂抹比一次大范围涂抹更准确

## 🐛 故障排除

### AI 融合失败
**症状**: 返回普通合成结果，无融合效果

**可能原因**:
1. 未配置 `OPENROUTER_API_KEY`
2. API 额度不足或网络问题
3. 图片过大导致超时

**解决方案**:
- 检查环境变量配置
- 查看终端日志中的错误信息
- 尝试使用较小的图片

### 融合效果不理想
**症状**: 光影不匹配、边缘生硬

**可能原因**:
1. 源图和目标图风格差异太大
2. 光照方向完全相反
3. 元素边缘质量差

**解决方案**:
- 选择风格相似的图片
- 尝试调整元素裁剪
- 使用 AI Generate 模式作为替代

### 处理时间过长
**症状**: 超过 60 秒仍在处理

**可能原因**:
1. 图片分辨率过高
2. API 响应慢
3. 网络连接问题

**解决方案**:
- 降低图片分辨率（建议 2048x2048 以下）
- 检查网络连接
- 查看 API 服务状态
